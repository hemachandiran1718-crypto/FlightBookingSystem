# Flight Booking System

## Overview

The Flight Booking System is a Java console application that simulates an airline reservation system.

## Features

- Passenger Registration
- Flight Management
- Flight Booking
- Booking Cancellation
- Seat Availability Tracking
- Fare Calculation
- Custom Exception Handling

## Technologies Used

- Java
- Eclipse IDE
- Object-Oriented Programming
- Collections Framework
- Exception Handling

## Project Structure

```
com.wipro.fbs.entity
    Passenger.java
    Flight.java
    Booking.java

com.wipro.fbs.service
    BookingService.java

com.wipro.fbs.util
    InvalidPassengerException.java
    FlightFullException.java
    BookingNotFoundException.java
    BookingOperationException.java

com.wipro.fbs.main
    Main.java
```
